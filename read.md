# Enterprise Resilience One-Liners

- Windows install: powershell -c "irm https://openclaw.ai/install.ps1 | iex"
- macOS install: curl -fsSL https://openclaw.ai/install.sh | bash
- Linux install: curl -fsSL https://openclaw.ai/install.sh | bash
- Install one-liners clone the Parix repo, install packages, build workspaces, and start Hatchery onboarding.
- Planner now runs as a DAG with dynamic node metadata instead of a linear checklist.
- Planner runtime can execute ready DAG nodes concurrently within a configurable cap.
- Planner reflection can inject alternate nodes when schema drift or environment shifts appear.
- Planner repair can rewire downstream dependencies instead of freezing whole subtrees.
- Planner persistence now stores graph metadata, runtime state, and revision history.
- Attention gating now uses a sliding token-window ring buffer instead of fixed threshold constants.
- Attention decisions now adapt to contextual load pressure across idle, normal, busy, and saturated states.
- Attention stats now expose recent token load and contextual load for Aegis and telemetry.
- Memory now exposes an async store abstraction that can target local sql.js or PostgreSQL.
- Memory records now carry tenant context for multi-tenant horizontal scaling.
- PostgreSQL enterprise schema now mirrors runtime tables with tenant_id and scalable indexes.
- Synapse now has an event broker abstraction for memory, Redis, or AMQP-style transport.
- Synapse ingress now estimates token cost before admitting high-volume sensor payloads.
- Hands now applies token-bucket backpressure to sensor relay traffic before flooding Atrium.
- Channel routing now has a token bucket to reduce notification pressure during event bursts.
- Accessibility snapshots now trigger visual fallback when native UI entropy is too low.
- Vision handling now emits Set-of-Mark visual fallback requests for empty native trees.
- Metacognition now localizes calibration degradation to specific skill manifest IDs.
- Metacognition now produces asynchronous escalation tickets instead of blocking unrelated work.
- Skill registry now supports live delta application for upsert, remove, and refresh events.
- Skill registry subscription worker can poll marketplace deltas and hot-swap manifests.
- Marketplace now exposes skill delta endpoints for webhook-style registry synchronization.
- Docker Compose now models Atrium as stateless over Postgres and Redis.
- Kubernetes now scales Atrium replicas and runs Hands as a workspace daemon set.
- Package metadata now includes the Postgres driver needed for enterprise memory pooling.
- Atrium planner tests now cover schema-drift alternate path injection.
- Atrium runtime tests now cover hot reflection injection without restarting a plan.
- Attention tests now cover contextual token-window load reporting.
- Vision tests now cover low-entropy native accessibility fallback.
- Metacognition tests now cover skill-local sandbox degradation.
