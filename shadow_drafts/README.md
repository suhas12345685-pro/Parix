# shadow_drafts

Parix uses this folder for zero-footprint proactive work.

Unfinished lead lists, study plans, generated scripts, research notes, and draft
deliverables should stay here until they are complete enough for Suhas to
review. Completed work can then be promoted into the main workspace by an
explicit plan step.
