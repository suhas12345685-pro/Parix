Parix is a local-first desktop agent with a TypeScript brain, Python OS sensors, an accessibility-first perception layer, and the Aegis dashboard for live control.

Quickstart

Verified Windows Path

  1. Install Node.js 20.10+ and Python 3.12.
  2. Clone the repository and install Node/Python dependencies.
  3. Run `npm run onboarding:web` and create a local profile.
  4. Open Aegis at `http://localhost:3000` and verify Hands is connected.
  5. Run `npm run test:e2e` before any demo or release rehearsal.

Skills

Local Skill Registry

Skills live in `.agents/skills//` with a `SKILL.md` manifest and optional scripts, references, and templates. The v0.2 registry is first-party and local.

Part Purpose
`SKILL.md` Human-readable trigger rules and workflow steps.
`scripts/` Reusable executors for repeatable operator work.
`references/` Small, task-specific context files.
`templates/` Reusable output shapes for reports and configs.

Accessibility moat

macOS

AX API backend is code-complete and awaiting real hardware verification.

Linux

AT-SPI2 backend is code-complete and awaiting distro verification.

Security model

Local by Default, Opt-In by Contract

  • Telemetry is disabled unless the profile has explicit consent.
  • Crash reports are sanitized and queued locally before optional upload.
  • Synapse binds to localhost by default.
  • Aegis exposes pause/resume and audit visibility.
