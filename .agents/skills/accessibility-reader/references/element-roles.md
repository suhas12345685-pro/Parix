# UI Element Roles Reference

## Interactive (can be acted upon)

| Role | Description | UIA Pattern |
|---|---|---|
| button | Clickable button | Invoke |
| edit | Text input field | Value |
| checkbox | Toggle on/off | Toggle |
| radiobutton | Select one of many | Toggle |
| combobox | Dropdown selector | Value + Expand |
| menuitem | Menu entry | Invoke |
| link / hyperlink | Clickable link | Invoke |
| listitem | List entry | SelectionItem |
| treeitem | Tree node | SelectionItem + Expand |
| tabitem / tab | Tab button | SelectionItem |
| slider | Range selector | RangeValue |
| spinner | Numeric up/down | Value |

## Container (layout, not directly interactive)

| Role | Description |
|---|---|
| pane | Generic container |
| group | Logical grouping |
| window | Top-level window |
| toolbar | Button strip |
| menu | Menu container |
| list | List container |
| tree | Tree container |
| tabcontrol | Tab strip |
| statusbar | Status line |

## Read-Only

| Role | Description |
|---|---|
| text | Static text label |
| image | Image element |
| progressbar | Progress indicator |
| tooltip | Hover text |
| titlebar | Window title |
